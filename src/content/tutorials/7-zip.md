## 下载与安装

1. 打开官网 **7-zip.org**，点击 "Download" 页面。
2. 选择对应系统版本：
   - 64 位 Windows：下载 `7z2401-x64.exe`（推荐大多数用户）。
   - 32 位 Windows：下载 `7z2401-x86.exe`（旧电脑）。
   - ARM64 Windows：下载 `7z2401-arm64.exe`（Surface Pro X 等 ARM 设备）。
3. 运行安装程序，选择安装目录（默认即可），点击 Install。
4. 安装完成后，右键菜单会多出一列 **7-Zip** 选项。

> **包管理器安装**：winget install 7zip.7zip（Windows 11 可用）。

安装包大小仅 **1.5MB**，是所有压缩工具中最轻量的。

## 基本使用

### 解压文件

**方法一：右键菜单（最快）**
1. 右键点击压缩文件（.zip/.rar/.7z 等）。
2. 选择 **7-Zip → 解压到当前文件夹**。
3. 文件直接解压到当前目录。

**方法二：打开程序操作**
1. 双击压缩文件，默认会用 7-Zip 打开。
2. 在 7-Zip 窗口中选中要解压的文件/文件夹。
3. 点击工具栏的"提取"按钮，选择目标路径。

### 压缩文件

**方法一：右键菜单**
1. 右键点击文件或文件夹。
2. 选择 **7-Zip → 添加到压缩包**。
3. 在弹出的窗口中设置参数：
   - **压缩格式**：7z（压缩率最高）、ZIP（兼容性最好）。
   - **压缩等级**：Ultra（最大压缩，速度慢）、Normal（均衡）、Fast（快速）。
   - **压缩方法**：LZMA2（7z 默认）、Deflate（ZIP 默认）。
4. 点击确定开始压缩。

**方法二：拖拽到已打开的程序**
1. 打开 7-Zip 程序（先不打开压缩包）。
2. 将文件拖拽到 7-Zip 窗口。
3. 会提示"添加到压缩包"，设置参数即可。

## 压缩格式详解

| 格式 | 压缩率 | 速度 | 兼容性 | 适用场景 |
|------|--------|------|--------|---------|
| **7z** | 最高 | 较慢 | 一般 | 自用存档，追求最小体积 |
| **ZIP** | 中等 | 快 | 最好 | 分享给他人，Windows 原生支持 |
| **TAR** | 无压缩 | 极快 | 好 | Linux 打包，通常配合 GZIP/XZ |
| **GZIP** | 中等 | 较快 | 好 | Linux 常用压缩格式 |
| **XZ** | 高 | 慢 | 一般 | 追求高压缩率的 Linux 场景 |
| **BZIP2** | 中高 | 较慢 | 较好 | 压缩大文本文件 |

日常使用建议：**自用选 7z，分享选 ZIP**。

## 加密与安全

### 加密压缩包

1. 右键文件 → 7-Zip → 添加到压缩包。
2. **加密**区域：
   - 输入密码（建议 8 位以上，包含字母和数字）。
   - **加密方法**：选择 AES-256（行业标准，安全可靠）。
   - 勾选"加密文件名"：别人看不到压缩包里有哪些文件（仅 7z 格式支持）。
3. 点击确定。
4. 解压时需要输入密码，否则无法打开。

> **安全提示**：7-Zip 使用 AES-256 加密，这是银行级别的加密标准。即使压缩包被他人获取，没有密码也几乎不可能破解。

### 批量处理

7-Zip 支持命令行操作，非常强大。以下是一些常用命令：

打开命令提示符（cmd）或 PowerShell：

```
# 把当前目录所有文件压缩为 test.7z
7z a test.7z *.*

# 解压 test.7z 到当前目录
7z x test.7z

# 加密压缩，密码为 123456
7z a -p123456 -mhe=on secret.7z *.*

# 查看压缩包内容
7z l test.7z

# 测试压缩包完整性
7z t test.7z
```

### 批量解压

如果有几十个压缩包需要解压，手动操作太慢：

1. 在文件夹中按 `Ctrl+A` 全选所有压缩包。
2. 右键 → 7-Zip → **解压每个压缩包到单独的文件夹**。
3. 批处理会自动解压，每个压缩包的内容放到以压缩包命名的文件夹中。

## 文件校验

7-Zip 可以计算文件的哈希值（校验码），用于验证文件完整性：

1. 右键文件 → **CRC SHA → SHA-256**（或其他）。
2. 计算完成后显示哈希值。
3. 与官方提供的哈希值对比，如果一致说明文件完整未被篡改。

## 常见问题

### 7-Zip 能解压 RAR 文件吗？

**可以。** 7-Zip 支持解压 RAR 文件（基于 unRAR 库）。但不能创建 RAR 格式（RAR 是专有格式，需要 WinRAR 才能创建）。

### 为什么 7-Zip 的界面这么简陋？

7-Zip 的开发者 Igor Pavlov 专注于功能和性能，不花时间在界面上。如果你在意界面，可以考虑搭配 **7-Zip Theme Manager** 换皮肤，或者直接通过右键菜单使用，完全不看主界面。

### 压缩后文件反而变大了？

一些已经压缩过的文件（如 JPG 图片、MP4 视频、MP3 音频）再次压缩时体积变化不大，甚至可能变大。这是正常的，因为这类文件已经经过了有损压缩。7-Zip 对**文本、代码、文档、配置文件**的压缩效果最好。

### 右键菜单没有 7-Zip 选项？

在 7-Zip 程序中：**工具 → 选项 → 插件**，确保 "7-Zip" 已启用。或者重新安装时勾选"集成到文件右键菜单"。

### 7-Zip 有 Mac 版吗？

有命令行版本（p7zip），但没有官方 GUI 版本。Mac 用户推荐系统自带的归档实用工具、The Unarchiver（免费），或 Keka（开源免费）。
