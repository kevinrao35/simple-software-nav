## 下载与安装

1. 打开官网 **blackmagicdesign.com/products/davinciresolve**。
2. 选择 **DaVinci Resolve (免费版)**，点击下载（约 3GB）。
3. Windows：运行 exe 安装程序，建议关闭其他程序再安装。
4. macOS：运行 dmg 安装包。
5. Linux：官方提供 .run 安装包。

### 系统要求（非常重要）

DaVinci Resolve 对硬件要求较高：

- **CPU**：建议 i7/锐龙 7 以上，8 核更佳。
- **内存**：**16GB 起步**，32GB 推荐。这是最低要求，8GB 会非常卡。
- **显卡**：建议独立显卡，NVIDIA GTX 1060 / AMD RX 580 以上。
- **存储**：建议 SSD，项目文件和缓存放在 SSD 上。
- **系统盘剩余空间**：至少 20GB 以上。

> **特别提示**：如果你的电脑是集成显卡或只有 8GB 内存，DaVinci Resolve 可能运行不流畅。建议先用 Shotcut 或 Olive 等轻量级剪辑软件过渡。

## 认识五大工作区

DaVinci Resolve 和 PR 最大的区别：**所有功能集成在一个软件里**。界面底部有五个标签页，代表五大工作区：

1. **剪辑（Cut & Edit）**：时间线剪辑、多机位编辑。
2. **Fusion**：视觉特效、动态图形（类似 After Effects）。
3. **调色（Color）**：好莱坞级别调色工具——这是 DaVinci 的看家本领。
4. **Fairlight**：音频编辑（类似 Audition）。
5. **交付（Deliver）**：导出设置。

## 剪辑入门：半小时剪出一个视频

### 新建项目

1. 启动 DaVinci Resolve，双击"新建项目"。
2. 输入项目名称，点击创建。
3. 进入**剪辑**工作区。

### 导入素材

- 直接拖拽视频/音频/图片到媒体池（左上角面板）。
- 或者点击媒体池的"导入素材"按钮。
- 支持几乎所有常见格式：MP4、MOV、AVI、MKV 等。

### 快速剪辑流程

1. 从媒体池拖拽素材到时间线。
2. 按 `I` 键设置入点，按 `O` 键设置出点。
3. 按 `F9` 将入出点之间的片段添加到时间线。
4. 用**切割工具**（`B` 键）在时间线上切割片段。
5. 用**选择工具**（`A` 键）拖拽调整片段位置。
6. 按 `Ctrl+B` 切割选中的片段。

### 常用剪辑快捷键

| 快捷键 | 功能 |
|--------|------|
| `I` | 设置入点 |
| `O` | 设置出点 |
| `F9` | 添加到时间线 |
| `B` | 切割工具 |
| `A` | 选择工具 |
| `Ctrl+B` | 切割选中的片段 |
| `Delete` | 删除片段 |
| `Shift+Delete` | 波纹删除（不留空隙） |
| `[` / `]` | 缩放时间线 |
| `Ctrl+Z` | 撤销 |
| `Shift+Z` | 适应时间线窗口 |

### 转场与标题

- **添加转场**：从效果库拖拽转场到两个片段之间。
- **添加标题**：效果库 → 标题 → 拖拽到时间线上方轨道。
- **修改文字**：在检查器（右上角面板）中修改内容和样式。

## 调色：DaVinci 的看家本领

### 进入调色工作区

点击底部的**调色**标签页。你会看到：
- 左侧：片段列表和画廊
- 中间：视频预览（分屏对比很有用）
- 右侧：调色工具面板

### 一级调色（基础校正）

1. **色轮（Color Wheels）**：
   - 提升暗部（阴影）、中间调、亮部（高光）。
   - 先调整对比度，让画面明暗分明。
2. **自动调色**：右键预览画面 → "自动调色"，AI 帮你做基础校正（很多时候效果不错）。
3. **色温/色调**：调整白平衡。
4. **饱和度**：增加或降低色彩鲜艳度。

### 二级调色（风格化）

1. **限定器（Qualifier）**：选中某个颜色范围单独调整。
   - 用吸管点击肤色，可以单独调整肤色，不影响背景。
2. **曲线（Curves）**：
   - 自定义曲线调整特定亮度范围或颜色通道。
   - 色相 vs 色相曲线：把蓝天变得更蓝，或把绿色调成青色调。
3. **LUT**：在节点上右键 → "应用 LUT"，可以加载专业调色预设。
   - 热门免费 LUT：Osiris、Buttery LUTs、M31 等。

### 节点工作流

DaVinci 用**节点**（Nodes）代替层级调整，每个节点是一个独立调整：

1. 右键预览画面 → **添加串行节点**，串联多个调整。
2. 右键选择 **添加并行节点**，同时应用多个调整。
3. 选中节点按 `Ctrl+D` 禁用/启用，对比效果。

> **调色思路**：先做一级校正（曝光/白平衡），再做风格化（创意调色），最后做局部调整（肤色/天空）。

## 音频处理

1. 切换到 **Fairlight** 工作区。
2. 在时间线上可以看到音频波形。
3. 调整音量：在片段上拖拽音量线，或在检查器中输入数值。
4. **添加音频效果**：
   - 压缩器（Compressor）：让音量更平稳。
   - EQ（均衡器）：调整频段，减少噪音或增强人声。
   - 降噪：效果库 → 音频插件 → 降噪。
5. **关键帧**：按住 `Alt` 点击音量线添加关键帧，制作淡入淡出。

## 导出设置

进入**交付**工作区：

1. **预设选择**：
   - 上传 B站/抖音 → H.264 Master（或自定义 MP4）。
   - YouTube 4K → YouTube 预设。
   - 保留最高画质 → QuickTime + ProRes 或 DNxHR。
2. **自定义设置**：
   - 格式：MP4（最通用）。
   - 编码：H.264 或 H.265。
   - 质量：选择"自动"或设定码率（推荐 20-40 Mbps）。
   - 分辨率：保持与素材一致。
3. 点击"添加到渲染队列"，然后点击"开始渲染"。

## 常见问题

### 免费版和 Studio 版有什么区别？

Studio 版（$295 一次性买断）比免费版多的功能：
- 10-bit H.265 编码支持
- 降噪（Neural Engine）
- 人脸识别
- 镜头畸变校正
- 胶片颗粒
- 远程协作

**对大多数创作者，免费版完全够用。**

### 剪辑时很卡怎么办？

1. **生成代理文件**：右键素材 → "生成代理媒体" → 选择 1080p 或 720p。
2. 代理模式下编辑，导出时自动替换为原素材。
3. **关闭调色**：在剪辑页面时，关掉预览窗口的调色显示。
4. **降低回放分辨率**：预览窗口旁选择 1/4 或 1/8 回放质量。

### 和 PR 比哪个好？

- **DaVinci Resolve**：调色无敌、集成度高、免费版慷慨。但学习曲线陡、吃配置。
- **Premiere Pro**：生态完善（AE/PS 联动）、插件多、行业标准。但贵（每月 ￥150+）。

建议：新手剪辑短视频用 **剪映**，想学专业剪辑从 **DaVinci Resolve** 开始。

### 导出文件太大？

H.265 编码比 H.264 体积小 50% 但画质相同。缺点是部分播放器需要解码支持。或者降低码率——在交付设置中手动输入码率数值。
