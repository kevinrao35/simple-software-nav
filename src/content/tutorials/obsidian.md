## 下载与安装

1. 打开官网 **obsidian.md**，点击 "Get Obsidian"。
2. 选择对应操作系统的版本下载。
3. 安装后启动，选择 **"创建一个新仓库"**（Vault）。
4. 给你的仓库取个名字，选择存放位置（建议放在同步网盘文件夹中）。
5. 创建完成，进入 Obsidian 界面。

Obsidian 和传统笔记软件最大的不同：**你的笔记就是本地文件夹里的纯文本 Markdown 文件**。这意味着即使没有 Obsidian，你也可以用记事本打开和编辑你的笔记。

## 核心概念：本地优先与 Markdown

### 为什么本地优先很重要

- **离线可用**：没有网络也能写。
- **数据所有权**：笔记存在你的硬盘上，不经过任何第三方服务器。
- **永不过时**：Markdown 是纯文本格式，任何编辑器都能打开。
- **无限自由**：可以用 Git 做版本管理，用同步盘多设备同步。

### Markdown 基础语法

在 Obsidian 中写笔记就是写 Markdown：

```
# 一级标题
## 二级标题
**加粗**
*斜体*
- 无序列表
1. 有序列表
[链接文字](https://xxx)
![图片描述](图片路径)
> 引用
`行内代码`
代码块就用三个反引号包裹
```

按 `Ctrl+P` 打开命令面板，搜索 "Markdown 格式参考" 查看更多。

## 双向链接：Obsidian 的灵魂

### 基础用法

- **创建链接**：输入 `[[` 开始输入笔记名称，自动搜索匹配的笔记，选中后按 Enter。
- **预览链接**：按住 `Ctrl` 点击链接，在新面板中打开。
- **未创建的笔记**：如果链接的笔记还没创建，链接会以特殊样式显示（带问号），点击即可新建。

### 反链面板

反链（Backlinks）是双向链接的核心——它告诉你**有哪些笔记链接到了当前笔记**：

- 在右侧面板打开"反链"视图。
- 看到所有引用了当前笔记的其他笔记。
- 这样当你修改某个想法时，所有关联的笔记都在眼前。

### 知识图谱

知识图谱是 Obsidian 最视觉冲击力的功能：

1. 点击左侧的"打开图谱"按钮（石墨烯图标）。
2. 每个圆点是一篇笔记，连线表示链接关系。
3. 可以 3D 旋转、缩放、筛选。
4. 点击任意节点跳转到对应笔记。

知识图谱的价值：**让零散的知识点变成一张网，发现你从未注意到的连接**。

## 插件生态：1500+ 种扩展方式

### 必装插件推荐

Obsidian 社区有超过 1500 款插件。以下是最值得安装的：

- **Calendar**：日历视图，点击日期快速创建"每日笔记"。
- **Kanban**：看板视图，用 Markdown 做项目管理。
- **Excalidraw**：在笔记中手绘流程图和白板。
- **Dataview**：把笔记变成数据库，用查询语言筛选笔记（进阶功能）。
- **Advanced Tables**：让 Markdown 表格编辑像 Excel 一样方便。
- **Templater**：高级模板系统，自动插入日期、时间、文件名。
- **Paste URL into selection**：选中文字后粘贴，自动生成带标题的链接。

### 如何安装插件

1. 点击左侧的"设置"按钮（齿轮图标）。
2. 选择 **社区插件 → 浏览**。
3. 搜索插件名称，点击"安装"。
4. 安装后需要在已安装插件列表中**启用**。

> **安全提示**：社区插件可以访问你的笔记内容。建议只安装评分高、更新活跃的插件。

## 主题系统：让笔记好看又护眼

1. 打开 **设置 → 外观 → 主题 → 管理**。
2. 浏览社区主题，找到喜欢的点击安装。
3. 热门推荐：**Minimal**（极简）、**Blue Topaz**（蓝色晶莹）、**AnuPpuccin**（柔和好配适）。
4. 安装后在主题下拉菜单中选择启用。

许多主题还支持分段颜色（Color Scheme），可以通过 **设置 → 外观 → 配色方案** 切换亮色/暗色。

## 每日笔记：建立写作习惯

### 设置每日笔记

1. 安装 Calendar 插件。
2. 点击日历上的日期自动创建当天的笔记。
3. 模板路径：设置 → 每日笔记 → 模板文件位置。

### 推荐模板

```
# {{date:YYYY-MM-DD}} {{date:dddd}}

## 📌 今日重点
- 

## 📝 工作记录
- 

## 💡 想法/灵感
- 

## 📖 阅读/学习
- 
```

## 发布到网站

Obsidian 付费服务 Obsidian Publish 每年 $96（约 700 元）。免费替代方案：

- **Obsidian 网页导出**：使用 **Webpage HTML Export** 插件将仓库导出为静态网站。
- **Quartz**：开源项目，将 Obsidian 仓库转换成静态博客网站，部署到 GitHub Pages 或 Vercel。
- **Mkdocs + Material**：如果你的笔记以文档为主，这是最好的导出方案。

## 常见问题

### Obsidian 免费够用吗？

个人笔记使用，核心功能全部免费。付费版本（Catalyst）主要是支持开发，不付费也不影响使用。

### 和 Notion 怎么选？

- 选 **Notion**：如果你需要团队协作、数据库功能、模板市场。
- 选 **Obsidian**：如果你在意数据隐私、喜欢 Markdown、想要知识图谱。

很多人两个一起用：Notion 做协作，Obsidian 做个人知识库。

### 换电脑后笔记怎么同步？

几种同步方案：
1. **Obsidian Sync**（官方付费方案，每月 $4）：端到端加密，最省心。
2. **iCloud**（Mac + iPhone 用户免费方案）。
3. **Syncthing**（免费开源，所有平台通用）。
4. **Git + GitHub**（免费，还自带版本管理）。

建议仓库目录放到同步盘里，这样手机和电脑都能编辑。
