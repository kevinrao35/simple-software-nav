## 下载与安装

1. 打开官网 **gimp.org**，点击首页的 "Download" 按钮。
2. 选择对应的操作系统版本（Windows / macOS / Linux）。
3. Windows 用户下载 exe 安装包，双击运行，一路 Next 即可完成安装。
4. macOS 用户下载 dmg 文件，拖拽到 Applications 文件夹。
5. Linux 用户建议通过包管理器安装：`sudo apt install gimp`（Ubuntu/Debian）或 `sudo dnf install gimp`（Fedora）。

安装完成后打开 GIMP，你会看到默认的三栏布局：左侧是工具箱，中间是画布，右侧是图层/通道/路径面板。

## 界面布局与基础设置

### 认识 GIMP 的三大区域

- **工具箱（左侧）**：包含所有工具——选区工具、画笔、橡皮擦、文字工具等。
- **画布（中间）**：编辑图像的主要工作区。
- **面板（右侧）**：图层、通道、路径、历史记录等管理面板。

### 推荐的首选项设置

打开 **编辑 → 首选项**，建议修改以下几项：

- **界面 → 主题**：选择深色主题（System），更护眼、更现代。
- **界面 → 图标主题**：选择 "Symbolic"，图标更清晰。
- **显示 → 缩放质量**：选择 "线性"，图像缩放更平滑。
- **输入设备**：如果你有数位板，在这里配置压感支持。

## 图层操作详解

### 创建和管理图层

图层是 GIMP 最核心的功能。你可以把图层想象成一张张透明的纸叠在一起。

- **新建图层**：`Ctrl+Shift+N` 或点击图层面板底部的"新建图层"按钮。
- **复制图层**：在图层面板右键图层 → "复制图层"。
- **删除图层**：选中图层后按 `Delete` 或点击面板底部的垃圾桶图标。
- **调整图层顺序**：在图层面板中拖拽图层上下移动。

### 图层蒙版

图层蒙版让你可以控制图层的哪些部分显示或隐藏：

1. 选中要添加蒙版的图层。
2. 右键图层 → "添加图层蒙版"。
3. 选择 "白色（完全不透明）" 作为初始蒙版。
4. 用黑色画笔在蒙版上涂抹，涂抹区域会变为透明。
5. 用白色画笔涂抹可以恢复显示。

> **小技巧**：按 `D` 键将前景色/背景色重置为黑白，按 `X` 键切换前景色和背景色。在蒙版编辑时非常实用。

### 混合模式

图层面板顶部的下拉菜单是混合模式选择器。常用模式：

- **正片叠底（Multiply）**：让图层变暗，适合叠加线条稿。
- **滤色（Screen）**：让图层变亮，适合叠加光效。
- **叠加（Overlay）**：增强对比度，适合纹理叠加。
- **柔光（Soft Light）**：温和的对比增强，适合调色。

## 选区工具使用指南

### 五种选区工具

- **矩形/椭圆选择**：`R` / `E` 键，最基础的选区工具，适合规则形状。
- **自由选择（套索）**：`F` 键，手动绘制选区边界。
- **模糊选择（魔棒）**：`U` 键，点击选择颜色相近的区域。**容差**值控制相似度范围。
- **按颜色选择**：`Shift+O`，选择图像中所有相似颜色的区域。
- **前景色选择**：智能选择前景物体，适合抠图。

### 选区操作

- **添加选区**：按住 `Shift` 键的同时使用选区工具。
- **减去选区**：按住 `Ctrl` 键的同时使用选区工具。
- **相交选区**：按住 `Shift+Ctrl` 同时使用选区工具。
- **反选**：`Ctrl+I`，反转当前选区。
- **取消选区**：`Ctrl+Shift+A`。
- **羽化边缘**：**选择 → 羽化**，让选区边缘柔和过渡。

## 照片修图实战

### 基础调色流程

1. **裁剪与构图**：使用裁剪工具（`Shift+C`）进行二次构图，遵循三分法原则。
2. **调整色阶**：**颜色 → 色阶**（`Ctrl+L`），拖动直方图两端的黑白滑块到数据起始处，让画面明暗范围更完整。
3. **调整曲线**：**颜色 → 曲线**，S 型曲线增强对比度。
4. **调整色相/饱和度**：**颜色 → 色相-饱和度**（`Ctrl+U`），提升色彩鲜艳度。
5. **锐化**：**滤镜 → 增强 → 锐化（USM）**，适量锐化让照片更清晰。

### 去除瑕疵

- **仿制图章工具**（`C` 键）：按住 `Ctrl` 点击要复制的源区域，然后在瑕疵处涂抹。
- **修复工具**：类似仿制图章，但会自动匹配周围纹理和光照，效果更自然。
- **模糊/锐化工具**：自由涂抹，局部模糊背景或锐化眼睛。

## 文字与排版

### 添加文字

1. 按 `T` 键选择文字工具。
2. 在画布上点击，输入文字。
3. 在工具箱底部的选项栏中调整字体、大小、对齐方式。
4. 文字会自动生成独立的文字图层，方便随时编辑。

### 文字特效

- **文字阴影**：复制文字图层，将下层文字填充为黑色，稍微偏移。
- **文字描边**：右键文字图层 → "Alpha 到选区"，然后 **编辑 → 描边选区**。
- **渐变文字**：创建渐变填充图层，右键 → "剪贴蒙版"，渐变会限制在文字范围内。

## 导出与保存

- **保存项目文件**：`Ctrl+S`，保存为 `.xcf` 格式（GIMP 的原生格式，保留所有图层信息）。
- **导出为图片**：
  - 使用 **文件 → 导出为**（`Ctrl+Shift+E`）。
  - 常用的导出格式：
    - **JPEG (.jpg)**：照片和复杂图像，可调节质量（85-95% 是较好的平衡）。
    - **PNG (.png)**：需要透明背景时使用。
    - **WebP**：网页用，体积小质量好。
- **批量导出**：**文件 → 批处理**（或使用 BIMP 插件），可以一次处理整个文件夹的图片。

## 常见问题

### GIMP 能代替 Photoshop 吗？

对于 80% 的日常场景——修图、做海报、设计素材、照片处理——GIMP **完全够用**。只有在非常专业的印刷出版（CMYK 色彩管理）或复杂的 PSD 特效文件方面会有限制。

### 界面太旧了怎么办？

GIMP 2.10 版本已经支持深色主题。打开 **编辑 → 首选项 → 界面 → 主题**，选择深色主题，配合 Symbolic 图标集，界面会现代很多。

### 找不到某个功能？

GIMP 的功能命名和分类和 Photoshop 不太一样。可以用 **帮助 → 搜索**（`Ctrl+F`）直接搜索功能名称，快速定位。

### 怎么装插件？

1. 下载插件文件（通常是 .py 或 .scm 文件）。
2. 复制到 GIMP 的用户插件目录：
   - Windows：`C:\Users\你的用户名\AppData\Roaming\GIMP\2.10\plug-ins`
   - macOS：`~/Library/Application Support/GIMP/2.10/plug-ins`
   - Linux：`~/.config/GIMP/2.10/plug-ins`
3. 重启 GIMP 即可生效。
