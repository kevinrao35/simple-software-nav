## 关于 OpenCut

OpenCut 是一个开源的视频编辑器，定位是 CapCut（剪映）的免费替代品。采用 MIT 开源协议，代码完全公开。

### 当前状态

OpenCut 正在进行架构重写，新版本采用 **Rust 核心 + 跨平台架构**。当前线上运行的版本是 classic 版，可通过 **opencut.app** 直接使用。

## 使用方式

### 网页版（无需安装）

直接打开 **opencut.app** 即可使用，浏览器打开就能开始剪辑。

### 桌面版

- Windows / macOS / Linux：从 GitHub Releases 下载对应安装包
- 安装后本地运行，所有处理在本地完成

## 核心特点

- **多轨道时间线** — 视频、音频、字幕多轨叠加编辑
- **跨平台** — 一套代码覆盖 Web、桌面、移动端
- **本地处理** — 不上传云端，保护隐私
- **MIT 协议开源** — 代码透明，可自由使用和修改
- **无水印导出** — 导出视频无水印、无分辨率限制

## 正在开发中的功能

OpenCut 新架构正在开发以下功能（来自官方 Roadmap）：

- **Editor API** — 编辑器 API
- **插件架构** — 支持第三方插件
- **MCP Server** — AI 助手集成
- **Headless 模式** — 自动化批量渲染
- **脚本面板** — 直接在编辑器中写脚本

## 常见问题

### OpenCut 和剪映哪个好？

OpenCut 优势在于完全开源免费、无水印、本地处理隐私安全。但目前功能成熟度不如剪映——剪映有更丰富的素材库和模板。如果在意隐私和开源，选 OpenCut；如果需要大量模板素材，剪映更方便。

### 支持哪些导出格式？

目前导出选项以网页版实际显示为准，不同版本可能有所不同。一般支持 MP4 格式导出。

### 需要注册账号吗？

不需要，OpenCut 无需注册即可使用。

### 项目还在开发中，稳定吗？

线上 classic 版基本稳定可用。新架构版本正在重写中，功能和性能会逐步提升。
