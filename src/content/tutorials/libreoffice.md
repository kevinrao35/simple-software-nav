## 下载与安装

1. 打开官网 **libreoffice.org**，点击首页的 "Download Now"。
2. 选择操作系统版本，下载安装包（约 300MB）。
3. Windows 用户运行 exe 安装包，选择 "Standard" 安装即可。
4. macOS 用户挂载 dmg 文件，将 LibreOffice 拖入 Applications 文件夹。
5. Linux 用户：`sudo apt install libreoffice`（Ubuntu）或 `sudo dnf install libreoffice`（Fedora）。

安装完成后，你会在开始菜单/应用程序列表中找到 6 个组件和 1 个快捷启动中心。

## Writer：文档处理完整指南

### 基本操作

- **新建文档**：`Ctrl+N`，或启动 LibreOffice Writer 自动创建空白文档。
- **保存文档**：`Ctrl+S`，默认保存为 .odt 格式。如果需要发给 Office 用户，选择 **文件 → 另存为 → Word 97-2003 (.doc)** 或 **Word (.docx)**。
- **打开 Word 文件**：直接拖拽 .doc/.docx 文件到 Writer，或者用 `Ctrl+O` 打开。

Writer 对 .docx 的兼容性在开源办公软件中是最好的，95% 以上的排版可以保留。

### 样式与排版

**样式的正确使用**——这是 Writer 最强大的功能之一：

1. 按 `F11` 打开样式面板。
2. 选中文字后双击样式名称即可应用：
   - 标题用 **"标题 1"、"标题 2"、"标题 3"** 层级。
   - 正文用 **"默认段落样式"**。
   - 列表用 **"列表"** 系列样式。
3. 修改样式后，所有应用该样式的内容会自动更新。

> **核心技巧**：不要手动调整字体和字号。统一修改样式 → 全文自动更新。写论文、做标书时这个功能极其高效。

### 目录自动生成

1. 为章节标题应用"标题 1"、"标题 2"这种层级样式。
2. 在文档开头点击 **插入 → 目录与索引 → 目录、索引或参考文献**。
3. 选择"目录"，点确定。目录自动生成。
4. 修改内容后，右键目录 → **更新目录**，一键刷新。

### 高级功能

- **邮件合并**：**工具 → 邮件合并向导**，可以从电子表格或数据库导入收件人列表，批量生成信函、标签。
- **脚注/尾注**：**插入 → 脚注和尾注**，写论文必备。
- **交叉引用**：**插入 → 交叉引用**，可以引用图表编号、章节标题，修改后自动更新。
- **文档比较**：**编辑 → 比较文档**，对比两个版本的差异。

## Calc：电子表格深度指南

### Excel 文件兼容

Calc 可以直接打开和编辑 .xls/.xlsx 文件，**数据透视表**和大部分**函数公式**兼容性良好。

### 数据透视表

1. 选中数据区域，点击 **数据 → 数据透视表 → 创建**。
2. 拖拽字段到行/列/数据区域。
3. 双击数据区域可以查看汇总明细。

### 条件格式

1. 选中要应用的数据区域。
2. **格式 → 条件格式 → 条件**。
3. 设置规则，比如："单元格值 > 100 → 绿色背景"。

### 函数公式

Calc 支持 500+ 函数，最常用的几类：

- **统计**：SUM（求和）、AVERAGE（平均）、COUNTIF（条件计数）
- **查找**：VLOOKUP（垂直查找）、MATCH、INDEX
- **逻辑**：IF（条件判断）、AND、OR
- **文本**：LEFT、RIGHT、MID、CONCATENATE

> **小技巧**：按 `Ctrl+Shift+F4` 打开函数向导，可以搜索和查看每个函数的用法。

### 百万行数据处理

如果数据量超过 10 万行，Excel 明显会卡顿。Calc 在处理百万行数据时表现更流畅。建议搭配筛选功能使用：

- 选中表头 → **数据 → 自动筛选**，按条件筛选数据。
- 使用 **数据 → 排序** 对数据进行排序。

## Impress：演示文稿制作

### 兼容 PPT 文件

Impress 可以直接打开和编辑 .ppt/.pptx 文件。复杂动画和过渡效果兼容性大约在 80% 左右。

### 模板使用

- 启动时选择"从模板创建"，可以看到内置的演示模板。
- 去 **extensions.libreoffice.org** 搜索 "Impress template"，可以下载大量免费模板。
- **slidesgo.com** 等第三方网站也有大量免费 PPT 模板，下载后可以在 Impress 中使用。

### 动画与切换

- **幻灯片切换**：点击右侧面板的"幻灯片切换"选项卡，为每页设置过渡效果。
- **自定义动画**：选中元素 → **幻灯片放映 → 自定义动画**，可以设置进入/退出/强调动画。

### 导出与分享

- **导出 PDF**：**文件 → 导出为 → 导出为 PDF**。支持 PDF 书签（从标题样式自动生成）。
- **导出图片**：**文件 → 导出**，选择 PNG 或 JPEG。
- **导出为 PPTX**：**文件 → 另存为 → Microsoft PowerPoint (.pptx)**。

## Draw：绘图功能详解

Draw 是很多人不知道的隐藏神器：

- **流程图**：使用连接器工具，自动吸附对齐，比 Visio 轻量得多。
- **海报制作**：Draw 的页面排版功能比 Writer 灵活，适合做单页海报。
- **导出 SVG**：绘图可以导出为矢量格式，无限放大不失真。

## Base：数据库入门

Base 可以连接多种数据库：

- 创建本地数据库（类似 Access）。
- 连接到 MySQL/PostgreSQL 等外部数据库。
- 自动生成表单，方便录入数据。

## Math：公式编辑器

在 Writer 中插入公式：

1. **插入 → 对象 → 公式**。
2. 在公式编辑器中输入 LaTeX 风格的代码，如 `a over b` 生成分数。
3. 也可以在公式面板中点选符号来插入。

## 跨平台同步技巧

LibreOffice 本身没有内置云同步，但可以通过以下方式实现：

- 将文档保存在 **OneDrive / Google Drive / 夸克网盘** 本地同步文件夹中。
- 配合 **Syncthing** 等开源同步工具实现多设备文件同步。
- 使用 **Nextcloud** 自建私有云，LibreOffice 支持 WebDAV 协议直接打开远程文件。

## 常见问题

### LibreOffice 和 OnlyOffice 哪个好？

LibreOffice 离线功能更强、组件更全；OnlyOffice 在线协作体验更好。建议日常使用 LibreOffice，需要多人协作时可以用 OnlyOffice 或 Google Docs。

### Writer 排版错位怎么办？

如果从 Word 打开的文档排版错位，尝试以下方法：
1. **文件 → 另存为 → 过滤器选择 "Word 2007-365 (.docx)"**。
2. 如果还是不理想，回到 Word 中保存为 .doc（97-2003 格式）再打开。
3. 复杂表格可以复制粘贴到 Calc 中处理，再贴回 Writer。

### 界面语言怎么切换？

**工具 → 选项 → 语言设置 → 用户界面**，选择简体中文，重启即可。
