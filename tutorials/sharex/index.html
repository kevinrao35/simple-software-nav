ShareX 使用教程：最强免费截图工具完全指南

入门软件：ShareX

下载与安装

1 打开官网 getsharex.com 或 GitHub 搜索 ShareX 下载。

2 下载安装版（推荐）或便携版（Portable）。

3 安装时选择"安装后启动"，首次启动会显示设置向导。

4 建议在向导中测试截图功能，确认快捷键生效。

ShareX 仅支持 Windows 平台。如果你是 macOS 用户，推荐用 Shottr 或 CleanShot X。

截图方式全解析

### 11 种截图方式

按 Print Screen（或你设置的快捷键）打开截图菜单，选择：

全屏截图：截取所有显示器内容。

活动窗口截图：自动识别当前激活窗口。

活动显示器截图：截取指定显示器。

窗口菜单选择：从列表中选窗口截图。

区域截图：拖拽选择任意区域。

区域（轻量）：更简洁的区域选择。

区域（透明）：半透明遮罩辅助选区域。

上次区域：重复上一次的选区。

自定义区域：固定尺寸/比例截图，适合截统一大小的素材。

滚动截图：自动滚动网页或文档，截成长图——这是 ShareX 的王牌功能之一。

文本截图（OCR）：截取后自动识别文字内容到剪贴板。

### 截图后自动动作

截图完成后，ShareX 可以自动执行一系列动作，在 任务设置 → 截图后 中配置：

打开预览窗口编辑。

上传到图床（自动复制链接）。

保存到指定文件夹。

复制到剪贴板。

打印图片。

我推荐的配置：截图 → 打开预览编辑 → 自动保存到桌面 → 复制到剪贴板。

屏幕录制与 GIF 制作

### 录制屏幕

1 按 `Shift+Print Screen` 打开录制菜单。

2 选择 屏幕录制（FFmpeg）。

3 框选录制区域，点击"开始录制"。

4 录制中悬浮工具栏可以暂停/停止。

5 停止后自动保存为 MP4 文件。

### 录制 GIF

1 选择 GIF 录制（需要 FFmpeg）。

2 框选区域，点击开始。

3 停止后自动导出 GIF 文件。

4 可以在 应用设置 → GIF 质量 中调整平衡文件大小和画质。

> 小技巧：录制教程时先用区域截图框选好范围，再切换到 GIF 录制，这样录制的尺寸一致。

上传与图床

### 内置图床

ShareX 支持 30+ 上传目的地：

Imgur：默认配置，匿名上传，适合临时分享。

ImgBB：免费图床，注册后可管理图片。

七牛云、又拍云：国内速度快，需要自己配置。

SM.MS：免费图床，无需注册。

S3/阿里云OSS/腾讯COS：适合自建图床。

自定义上传：可以配置自己的服务器上传脚本。

### 配置自定义图床

以 SM.MS 为例：

1 上传目的地设置 → 添加 → 图像上传器 → SM.MS。

2 获取 API Key（去 sm.ms 注册获取）。

3 填入 API Key，测试连接。

4 上传后链接自动复制到剪贴板。

### 自动上传流程

最实用的工作流：截图 → 自动上传 → 链接复制到剪贴板 → 直接粘贴分享。

配置方法：任务设置 → 截图后，勾选"上传图片到图床"和"复制 URL 到剪贴板"。

内置编辑器功能

截图后自动弹出编辑器，功能比想象中丰富：

基础标注：矩形/椭圆/箭头/线条/文字——做教程截图必备。

高亮/模糊：突出重要内容，遮盖敏感信息。

马赛克：打码敏感区域。

步骤编号：自动排列数字序号，做操作步骤图很方便。

水印：批量添加水印（在设置中可配置默认水印）。

图像调整：裁剪、旋转、调整大小、翻转。

> 快捷键：编辑器中 Ctrl+Z 撤销，Ctrl+Shift+Z 重做，Esc 关闭并保存。

实用小工具

ShareX 还集成了十几个实用工具：

取色器：屏幕取色，显示 HEX/RGB/HSL 值，一键复制。

屏幕尺子：测量屏幕上任意区域的像素尺寸。

二维码识别：截图二维码区域，自动识别内容。

二维码生成：输入文字/链接，生成二维码。

图像合并：把多张图片纵向或横向拼合。

图像分割：把一张图按网格切分（做九宫格可以用）。

视频缩略图：从视频中抽取帧生成缩略图网格。

DNS 修改器：快速切换 DNS 设置。

颜色拾取：从屏幕任意位置拾取颜色。

文件哈希：计算文件的 MD5/SHA1/SHA256。

剪贴板查看：查看剪贴板历史。

快捷键配置

热键设置 → 快捷键 中可以自定义所有操作的快捷键。

建议配置：

| 功能 | 推荐快捷键 |

|------|-----------|

| 区域截图 | Print Screen |

| 滚动截图 | Ctrl+Print Screen |

| 屏幕录制 | Shift+Print Screen |

| GIF 录制 | Ctrl+Shift+Print Screen |

| 打开主窗口 | Ctrl+Shift+X |

| 上传剪贴板图片 | Ctrl+Shift+U |

| 取色器 | Ctrl+Shift+C |

常见问题

### 截图后在编辑器中看不到图片？

检查 任务设置 → 常规 中的"截图后"动作，确保"打开图片编辑器"已勾选。

### 上传失败/连接超时？

中国网络访问 Imgur 可能不稳定。建议：

1 更换为国内图床（SM.MS、七牛云）。

2 或在 上传目的地设置 → Imgur → 上传配置 中设置代理。

### 滚动截图只能截一小段？

滚动截图依赖于系统滚动速度。在 应用设置 → 高级 → 滚动截图 中：

增加"滚动延迟"（建议 500ms）。

减少"滚动步长"。

重新尝试时尽量让滚动区域处于激活状态。

### ShareX 和 Snipaste 哪个好？

Snipaste 的贴图功能（按 F1 截图后按 F3 贴在屏幕上）是独一无二的。但整体功能的丰富程度，ShareX 远胜。两者不冲突：可以 Snipaste 日常贴图，ShareX 做深度截图和上传。
