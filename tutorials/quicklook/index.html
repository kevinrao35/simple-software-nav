软件简介

QuickLook 是一款 Windows 平台的开源文件快速预览工具，相当于把 macOS 的 Quick Look（空格预览）功能带到了 Windows 上。安装后，选中任意文件按下空格键，无需打开对应的应用程序即可预览文件内容。预览窗口支持缩放、滚动、播放等交互操作，关闭窗口也只需按一下 Esc 或空格键即可。

这款软件由开发者 QL-Win 团队维护，基于 GPL-3.0 协议开源。相较于微软应用商店中需要付费或含广告的同类软件（如 Seer），QuickLook 完全免费且没有广告，插件系统也提供了丰富的扩展能力。

核心特性

空格即预览：选中文件按空格键，秒开预览窗口

格式广泛：支持图片、视频、音频、PDF、Office 文档、代码、压缩包等

交互操作：预览窗口中支持缩放、旋转、播放、翻页

插件系统：通过 .qlplugin 插件文件扩展更多格式支持

轻量无驻留：不使用时几乎不占用系统资源

开源免费：GPL-3.0 协议，无广告无收费

安装方法

### 方法一：GitHub Releases 下载（推荐）

1 访问 https://github.com/QL-Win/QuickLook/releases

2 下载最新版本的安装包（如 `QuickLook-4.0.0.msi`）

3 双击运行安装程序，按向导完成安装

4 安装完成后，QuickLook 会自动启动并驻留在系统托盘中

### 方法二：winget 安装

winget install QuickLook

### 依赖要求

QuickLook 依赖 Microsoft Visual C++ Redistributable。如果安装后按空格键无反应，这是最常见的原因。解决方法：

1 访问微软官网下载 Visual C++ Redistributable：

- https://aka.ms/vs/17/release/vc_redist.x64.exe

2 下载并安装 `vc_redist.x64.exe`

3 重启 QuickLook（右键系统托盘图标 > 退出，重新启动 QuickLook）

### 启动与关闭

安装后 QuickLook 会自动设置为开机自启。托盘区域会出现 QuickLook 图标（一个眼睛图标）。

右键点击托盘图标：可以退出、重启、进入设置

若要完全关闭：右键托盘图标选择退出，然后从启动项管理中禁用

支持的预览格式清单

QuickLook 原生支持以下文件格式的预览：

### 图片类

`.jpg` / `.jpeg` / `.png` / `.bmp` / `.gif` / `.webp` / `.ico` / `.tiff` / `.svg` / `.psd` / `.dds` / `.cr2`（佳能 RAW 格式）

### 文档类

`.pdf`（使用内置 PDF 渲染器）

Office 系列：`.docx` / `.xlsx` / `.pptx`（需要系统已安装 Office 2007 或更高版本）

纯文本：`.txt` / `.log` / `.ini` / `.cfg`

Markdown：`.md` / `.markdown`（预览为渲染后的 HTML 样式）

代码文件：`.py` / `.js` / `.ts` / `.html` / `.css` / `.json` / `.xml` / `.yaml` / `.yml` / `.cpp` / `.c` / `.h` / `.java` / `.cs` / `.go` / `.rs` / `.rb` / `.php` / `.sh` / `.bat` / `.ps1` / `.sql` / `.swift` 等（语法高亮显示）

### 媒体类

视频：`.mp4` / `.avi` / `.mkv` / `.mov` / `.wmv` / `.flv` / `.webm` / `.m4v`

音频：`.mp3` / `.wav` / `.flac` / `.aac` / `.ogg` / `.wma` / `.m4a` / `.opus`

### 压缩包类

`.zip` / `.rar` / `.7z` / `.tar` / `.gz` / `.bz2`

### 其他

`.html`：预览渲染后的网页

`.dll` / `.exe`：显示文件属性信息

`.lnk`：显示快捷方式的目标路径

文件夹：按空格预览文件夹内部文件列表（浅度扫描）

基本操作

### 预览操作

| 操作 | 效果 |

|------|------|

| 选中文件，按 空格 | 打开/关闭预览窗口 |

| 预览中按 Esc | 关闭预览窗口 |

| 预览中按 空格 | 切换预览开关 |

| 预览中按 F1 | 使用默认程序打开该文件 |

| 预览中按 F5 | 进入全屏模式 |

| 预览中按 F11 | 全屏模式切换 |

| 预览中滚动 滚轮 | 缩放图片 / 翻页（PDF/文档） |

| 预览中 Ctrl + 滚轮 | 缩放内容 |

| 预览中 方向键 | 翻页（PDF/文档） / 上一个/下一个文件 |

| 预览中 鼠标点击空白区域 | 关闭预览窗口 |

### 连续预览技巧

在文件夹中选中一个文件预览后，按 上/下方向键 可以切换到同目录下的上一个/下一个文件并自动预览。这对于快速浏览大量图片或文档非常实用。

也可以用 鼠标滚轮 在文件列表切换到下一个文件。切换时预览窗口会保持打开。

### 图片/视频预览技巧

图片预览时滚动滚轮可以放大缩小，方便查看细节

视频预览时支持播放/暂停（空格键控制），进度条可以拖拽跳转

GIF 图片在预览窗口中会自动播放

插件系统

QuickLook 支持通过插件扩展更多文件格式的预览能力。

### 查找和下载插件

官方插件列表页面：https://github.com/QL-Win/QuickLook/wiki/Plugins

常用插件推荐：

| 插件 | 功能 |

|------|------|

| Office/Word | 预览 .doc、.xls、.ppt 等旧版 Office 格式 |

| CAD | 预览 .dwg 和 .dxf 文件 |

| Epub | 预览电子书文件 |

| IPynb | 预览 Jupyter Notebook 文件 |

| Apache POI | 预览 .doc、.xls、.ppt（不使用系统 Office 的独立方案） |

| RAW | 预览更多相机 RAW 格式 |

| Registry | 预览 .reg 注册表文件 |

| Shader | 预览着色器文件 |

### 安装插件

1 从插件开发者处下载 `.qlplugin` 文件

2 双击 `.qlplugin` 文件，QuickLook 会自动安装

3 或者打开 QuickLook 的设置窗口，在插件管理页面中点击 从文件安装

4 安装完成后可能需要重启 QuickLook 才能生效

### 卸载插件

1 右键点击系统托盘 QuickLook 图标

2 选择 设置 或 偏好设置

3 进入 插件管理 页面

4 选择要卸载的插件，点击 卸载

与 Seer 的比较

Seer 是 Microsoft Store 上的一款同类软件（售价约 19 元或 Freemium 模式），两者的主要区别：

| 对比项 | QuickLook | Seer |

|--------|-----------|------|

| 价格 | 完全免费 | 付费/Freemium |

| 开源 | 是（GPL-3.0） | 否 |

| 插件系统 | 支持 .qlplugin | 有限支持 |

| 格式支持 | 广泛 | 广泛 |

| 更新频率 | 较为活跃 | 更新较慢 |

| 资源占用 | 轻量 | 略高 |

| 稳定度 | 偶尔轻微闪烁 | 较稳定 |

| 商店版 | 无 | 有（Store 搜索 Seer） |

总体来说，QuickLook 在免费和开源方面有绝对优势，插件生态也更开放。如果遇到 QuickLook 的某些格式预览不稳定，Seer 可以作为备选方案。

排错指南

### 问题 1：按空格键没有任何反应

原因：最常见的原因是缺少 Visual C++ 运行库。

解决：

1 下载并安装 `vc_redist.x64.exe`（链接见上文安装部分）

2 重启 QuickLook

3 如果仍然无效，检查系统托盘中是否有 QuickLook 图标

4 如果没有图标，说明程序没有正常启动，尝试以管理员身份运行

### 问题 2：部分文件类型无法预览

原因：QuickLook 需要相应的后端程序支持某些格式（如 Office 文档需要安装 Office）。

解决：

1 Office 文档预览需要系统安装了 Office 2007 或更高版本

2 PDF 预览内置支持，如果无法预览可尝试安装 PDF 插件

3 安装对应格式的 QuickLook 插件

4 检查文件是否损坏

### 问题 3：预览窗口闪烁或卡顿

解决：

1 更新显卡驱动程序（特别是 Intel 核芯显卡）

2 在 QuickLook 设置中关闭硬件加速（如果有此选项）

3 降低系统显示缩放比例（100% 缩放时最稳定）

4 更新到最新版本的 QuickLook

### 问题 4：开机不自启

解决：

1 右键系统托盘 QuickLook 图标

2 检查是否有 开机启动 选项

3 如果没有，在系统的任务管理器 > 启动中手动添加 QuickLook

4 或者创建一个 QuickLook 的快捷方式放到 `shell:startup` 文件夹中

### 问题 5：预览 PDF 中文显示乱码

解决：

1 确保系统字体中包含文档使用的中文字体

2 安装微软雅黑等常用中文字体包

3 如果问题持续，尝试安装其他 PDF 阅读器作为默认程序

卸载方法

1 右键托盘 QuickLook 图标，选择退出

2 打开 设置 > 应用 > 应用和功能（Windows 10/11）

3 搜索 QuickLook，点击卸载

4 或者直接运行安装包，选择卸载选项
