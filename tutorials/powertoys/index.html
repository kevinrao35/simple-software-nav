安装方式

### 方式一：Microsoft Store（推荐）

打开 Microsoft Store，搜索 "Microsoft PowerToys"，点击安装。这是最简单的安装方式，支持自动更新。

### 方式二：winget 命令行

winget install Microsoft.PowerToys -s winget

适合习惯命令行的用户，安装后同样支持自动更新。

### 方式三：GitHub Releases

访问 [PowerToys Releases 页面](https://github.com/microsoft/PowerToys/releases)，下载最新版本的 PowerToysSetup-x.xx.x-x64.exe 安装包手动安装。

安装完成后，PowerToys 会自动在系统托盘中运行（任务栏右下角的小图标）。双击托盘图标或右键选择"设置"即可打开主界面。

最实用的 10 个工具详解

### 1. FancyZones —— 窗口布局管理

FancyZones 是 PowerToys 中最受欢迎的工具，可以让你将桌面划分为自定义的窗口布局区域，拖拽窗口到对应区域即可自动吸附。

使用方法：

1 在 PowerToys 设置中进入 FancyZones 页面，确保已启用（默认开启）。

2 点击 "启动布局编辑器"（或按 `Win+`` ` 快捷键）。

3 在布局编辑器中选择已有模板，或点击 "创建新布局" 自定义。

- 焦点布局：中间一个大区域 + 两侧小区域，适合主副窗口搭配。

- 网格布局：将屏幕划分为等大的网格（如 3x3 九宫格），适合多窗口排列。

- 行列布局：按行或列划分区域，适合对比文档或代码。

4 按住 `Shift` 键拖拽窗口到目标区域，窗口会自动吸附到该区域。

5 在设置中可调整区域间距（默认 8px），以及是否覆盖 Win+箭头键的默认贴靠行为。

推荐设置：勾选"在拖拽时按住 Shift 键来激活区域"，避免与系统贴靠冲突；间距设为 4~6px 用起来更紧凑。

截图描述：FancyZones 布局编辑器中，一个 3 列等宽的网格布局模板被选中覆盖在桌面上，每个区域用半透明蓝色高亮，底部有保存和取消按钮。

### 2. PowerRename —— 批量重命名

PowerRename 支持使用正则表达式或普通文本替换进行批量重命名，且提供实时预览，不怕改错。

使用方法：

1 在文件资源管理器中选中多个文件（`Ctrl+A` 全选或 `Ctrl+单击` 多选）。

2 右键点击任意选中文件，菜单中会出现 "PowerRename"（如果没有，请确认配置了鼠标右键菜单）。

3 在 PowerRename 窗口中配置：

- 搜索于：输入要查找的文本（如 photo）或正则表达式（如 \d{4} 匹配四位数字）。

- 替换为：输入替换后的文本（如 img），留空则为删除匹配内容。

- 使用正则表达式：勾选后启用正则匹配。

4 下方列表会实时显示每个文件的修改前名称和修改后名称，绿色 = 重命名、红色 = 无变化（不影响）。

5 确认无误后点击 "应用"。

常用正则示例：

| 目的 | 搜索于 | 替换为 |

|------|--------|--------|

| 删除文件名中的空格 | \s+ | - |

| 将数字改为 3 位编号 | \d+ | {Integer.Parse($0):D3} |

| 提取日期部分 | (\d{4})-(\d{2})-(\d{2}) | $1-$2-$3 |

> 正则重命名使用 .NET 正则引擎，$0 代表整个匹配，$1、$2 等代表捕获组。

截图描述：PowerRename 窗口顶部是搜索和替换输入框及选项（正则、忽略大小写、仅文件名等）；中部是预览列表，左侧修改前名称、右侧修改后名称的预览；底部是"应用"和"取消"按钮。

### 3. Color Picker —— 屏幕取色器

一键拾取屏幕上任意位置的颜色，支持 HEX、RGB、HSL 等多种格式自动复制到剪贴板。

使用方法：

1 按快捷键 `Win+Shift+C`（默认）激活 Color Picker。

2 鼠标指针变成放大镜，屏幕出现取色放大预览窗口，显示当前像素颜色。

3 移动鼠标到目标颜色位置，点击确认取色。

4 Color Picker 窗口弹出，显示该颜色的详细信息。

5 点击所需格式（HEX / RGB / HSL 等）右侧的复制按钮，或直接按 `Ctrl+C` 复制。

6 可以在编辑器历史中查看最近取色的记录。

设置建议：在 PowerToys 设置 > Color Picker 中可启用"激活时显示编辑器"（建议开启），或设置为"仅将颜色复制到剪贴板"（取色后直接复制，不弹窗）。

截图描述：激活 Color Picker 后屏幕中央出现一个圆形放大取色器，显示鼠标所在位置的放大像素网格，底部显示当前颜色的 HEX 值。

### 4. Text Extractor —— OCR 文字提取

可提取屏幕上任意区域的文字（包括图片、视频中的文字），基于 Windows OCR 引擎，无需联网。

使用方法：

1 按快捷键 `Win+Shift+T` 激活。

2 屏幕顶部出现操作提示条，鼠标指针变为十字线。

3 按住鼠标左键拖拽框选要提取文字的区域（类似截图）。

4 松开鼠标后，OCR 引擎自动识别选中区域中的文字。

5 识别后的文字自动复制到剪贴板，同时屏幕右下角弹出通知提示。

6 打开记事本或其他文本编辑器 `Ctrl+V` 粘贴即可。

注意事项：

OCR 识别的准确性取决于文字清晰度和字号，推荐不小于 12px 的文字。

不支持手写文字的识别。

如果识别结果为空，尝试选择更大区域包含更多上下文。

截图描述：Text Extractor 激活后屏幕顶部出现一个半透明的操作栏，包含"拖拽选择文字区域"的提示，鼠标变成十字线，用户正在拖拽框选一个包含文字的图片区域，选框呈半透明蓝色。

### 5. Keyboard Manager —— 键位映射与快捷键重定义

将任意按键映射为其他按键，或创建自定义快捷键组合。

使用方法：

1 在 PowerToys 设置中进入 键盘管理器。

2 重新映射键：如将 `Caps Lock` 映射为 `Ctrl`。

- 点击"重新映射键" > "+"添加新映射。

- 左侧选择或按键输入源键，右侧选择目标键。

3 重新映射快捷键：如将 `Ctrl+Shift+Escape` 映射为打开截图工具（`Win+Shift+S`）。

- 点击"重新映射快捷键" > "+"添加。

- 左侧填写原快捷键组合，右侧填写目标快捷键组合。

4 设置立即生效，无需重启。

实用映射建议：

`Caps Lock` → `Ctrl`（对 Vim/Emacs 用户非常友好）

`Ctrl+Alt+T` → 打开 CMD/终端

`Win+Q` → 关闭当前窗口（代替 Alt+F4）

截图描述：键盘管理器设置页，上半部分是"重新映射键"列表，显示将 Caps Lock 映射为 Ctrl 的条目；下半部分是"重新映射快捷键"列表，显示多个自定义快捷键映射。

### 6. PowerToys Run —— 快速启动器

类似于 macOS 的 Spotlight 或 Flow Launcher，按快捷键呼出搜索框，快速启动应用、搜索文件、计算表达式等。

使用方法：

1 按 `Alt+Space` 呼出 PowerToys Run 搜索框（可自定义快捷键）。

2 输入关键词即可搜索程序和文件，支持模糊匹配和拼音。

3 输入 `=` 开头可直接执行计算，如 `=1024*768`。

4 输入 `>` 开头可执行 Shell 命令，如 `>ipconfig`。

5 输入 `//` 开头可打开 URL，如 `//github.com`。

6 使用 `Tab` 和箭头键选择结果，`Enter` 打开，`Ctrl+Shift+Enter` 以管理员身份打开。

插件说明： PowerToys Run 支持插件扩展，默认启用系统命令、计算器、路径、Shell、Windows 设置、服务等插件。可在 PowerToys 设置 > PowerToys Run 中启用/禁用各插件。

截图描述：屏幕中央出现 PowerToys Run 的搜索框，搜索框下方显示匹配的应用和文件列表，搜索框中输入了"计算器"进行中文搜索，下方结果中显示了系统计算器和相关文件。

### 7. File Locksmith —— 文件占用检查

当删除或移动文件时提示"文件被占用"，File Locksmith 能告诉你哪个进程正在使用该文件。

使用方法：

1 在文件资源管理器中右键点击被占用的文件或文件夹。

2 选择菜单中的 "File Locksmith 解锁文件"。

3 弹窗列出所有占用该文件的进程，包括进程名称、PID 和路径。

4 点击进程可查看详细信息，点击"结束任务"直接终止进程。

截图描述：File Locksmith 弹窗，上半部分显示被检查的文件路径，下方列表显示占用进程（如 explorer.exe、notepad.exe），每个进程有结束任务按钮。

### 8. Mouse Without Borders —— 跨设备鼠标/键盘共享

使用一套鼠标键盘控制最多 4 台 Windows 电脑，类似硬件 KVM 切换器但不需要额外硬件。

使用方法：

1 在每台电脑上安装 PowerToys 并启用 Mouse Without Borders。

2 在设置中点击 "启动 Mouse Without Borders"。

3 连接向导会显示配对码（security key），在另一台电脑输入相同配对码即可连接。

4 连接成功后，鼠标移动到屏幕边缘会自动过渡到另一台电脑的屏幕。

5 可以使用拖拽实现跨设备文件复制（一次最大拖拽一个文件）。

6 支持跨设备剪贴板共享（文字和小于 10MB 的图片）。

要求： 所有设备必须在同一局域网内。

截图描述：Mouse Without Borders 设置页面，显示已连接的设备列表，每台设备用不同的颜色标记，设备排列成虚拟桌面布局，用户可以通过拖拽调整排列位置。

### 9. Peek —— 文件快速预览

选中文件后按快捷键快速预览内容，无需打开完整应用。类似 macOS 的 Quick Look（空格预览）。

使用方法：

1 在文件资源管理器中选中任意文件（图片、文档、视频、PDF 等）。

2 按快捷键 `Ctrl+Space`（默认）。

3 弹出预览窗口显示文件内容：

- 图片：大图预览 + 分辨率、大小信息。

- 文本/代码：语法高亮预览 + 行数统计。

- PDF：嵌入式预览。

- 视频/音频：嵌入式播放器，可直接播放。

4 再次按 `Ctrl+Space` 或按 `Esc` 关闭预览。

截图描述：Peek 预览窗口中显示一个 Markdown 文件，代码块带有语法高亮，窗口右上角显示文件大小和修改日期，底部有"用默认程序打开"按钮。

### 10. Advanced Paste —— 智能粘贴

将剪贴板中的内容转换为需要的格式（纯文本、Markdown、JSON 等）。

使用方法：

1 复制任意内容到剪贴板。

2 按 `Win+Shift+V` 呼出 Advanced Paste 对话框。

3 选择目标格式：

- 纯文本：去除所有格式。

- Markdown：将 HTML 格式转换为 Markdown 语法。

- JSON：将文本转为 JSON。

4 转换后的内容自动粘贴到当前光标位置。

> 此功能需要启用 AI 功能（可选），也可使用本地转换模式。

截图描述：Advanced Paste 窗口，上方显示原始剪贴板内容预览，下方有格式选择下拉菜单（纯文本 / Markdown / JSON），点击"粘贴"按钮后自动粘贴到目标位置。

常见问题

Q：PowerToys 支持 Windows 10 和 Windows 11 吗？

A：Windows 10（2004 及以上）和 Windows 11 均可安装。部分功能（如 Peek）在 Windows 11 上表现更佳。

Q：为什么某个功能没有效果？

A：请按以下步骤排查：

1 检查该功能是否已在 PowerToys 设置中启用（右上角开关）。

2 检查快捷键是否与其他软件冲突。

3 重启 PowerToys 进程（在设置页面底部点击"重启 PowerToys"）。

4 更新到最新版本。

Q：PowerToys 有性能影响吗？

A：只开启需要的功能即可。通常仅占用 100~200MB 内存，CPU 占用极低（闲置时接近 0%）。不建议同时开启全部工具。

Q：如何卸载 PowerToys？

A：进入 Windows 设置 > 应用 > 已安装的应用，搜索 "Microsoft PowerToys" 卸载。卸载后所有功能即停止，不影响系统本身。

Q：PowerToys 的开源协议是什么？

A：MIT 协议，可自由使用、修改和分发。源代码托管在 [GitHub](https://github.com/microsoft/PowerToys)。
